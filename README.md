# Dog Language Translator

A simple web app that translates dog behaviors into human-understandable meanings using JavaScript keyword matching.

# Features
- Chat-style interface
- Detects dog behaviors (barking, wagging, whining, etc.)
- Instant responses using simple logic
- Beginner-friendly JavaScript project

# How it works
User enters a dog behavior → JavaScript checks keywords → shows matching meaning.

## Tech Stack
- HTML
- CSS
- JavaScript
